# Valve Driver V1 Project

## Description
This project contains the schematic, PCB, and bill of materials for the Valve Driver V1 power/driver board designed to control a 12V valve with a maximum load of 1A.

## Release Notes
- Target: 12V valve up to 1A
- Topology: Low-side N-MOSFET, TC4427, TVS clamp at 33V, SS34
- Rsh = 50mΩ
- INA240 scaling for 3.3V MCU (Vout≈2.0V @1A)
